# Github AI Models

## '.env' file

```
GITHUB_TOKEN="..."
```

## Manage Conda ENV

### Create

```
conda env create -n python-setup-project-env -f ./env.yml
```

### Activate
```
conda activate python-setup-project-env
```

### Update

```
conda env update -n python-setup-project-env -f ./env.yml
```

### Remove

```
conda env remove --n python-setup-project-env
```

## Execute

```
python src/main.py
```
