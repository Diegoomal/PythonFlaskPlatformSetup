# Python simple project

## '.env' file

```
TEST=test
```

## Execute

```
conda activate project-env

conda env create -n project-env -f ./env.yml

python src/main.py
```

## Update env

```
conda env update -n project-env -f ./env.yml
```

## Remove env

```
conda env remove --n project-env
```
